\jtitle{線形モデルによるレビュアー個性を生かした映画推薦とその要因の評価}
\etitle{Personalized Movie Recommendation using Linear Models:\\
Reviewer-Aware Contribution Analysis}

\jauthor{神戸大学 \, {\small ○}渡邊 咲樹，村尾 元}
\eauthor{Watanabe Saki, Murao Hajime, Kobe University}

\englishabstract{
This study proposes an explainable movie recommendation model using a
group-wise linear regression framework. User ratings are decomposed into
four additive components, enabling stable and interpretable contribution
analysis. Experiments on an IMDb-based dataset confirm that the method
achieves practical accuracy while revealing consistent user evaluation tendencies.
}

\keywords{Recommender System, Movie, Explainability, Linear Model}
